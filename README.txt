# 🎬 Cadastro de Filmes com Streamlit + SQLite

Este é um projeto simples para cadastro e visualização de filmes utilizando **Python**, **SQLite** e **Streamlit**.

## 📦 Tecnologias utilizadas

- Python 3
- SQLite (banco de dados local)
- Streamlit (interface web)

## 📋 Funcionalidades

- ✅ Cadastro de filmes com nome, ano e nota
- 📄 Visualização em tabela de todos os filmes cadastrados
- 🗃️ Armazenamento persistente em banco de dados local (`titulo.db`)
